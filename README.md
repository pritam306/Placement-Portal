# 🚀 WB Placements Analysis Portal

A **data-driven web application** that analyzes student placement trends across **West Bengal institutions**. This platform provides interactive dashboards and insights to help students understand placement patterns and make better career decisions.

## ✨ Features

* Interactive dashboards (charts & graphs)
* Year-wise placement trend analysis
* Company-wise recruitment insights
* Department performance comparison
* (DYNAMIC PROGRAMMING) ML-based placement prediction
* Modern UI (Glassmorphism design)
* FireBase backend integrated

## Tech used

* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** Chart.js / Custom JS
* **Backend:** Python Firebase Superbase Openrouter
* **Tools:** Figma ApiFox VS code Ganalytics
