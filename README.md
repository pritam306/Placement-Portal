WB Placements Analysis is a data-driven web application that analyzes student placement trends across West Bengal institutions. The platform visualizes key insights such as placement rates, average salary packages, company-wise recruitment, and department performance using interactive charts and dashboards.

The project integrates frontend technologies (HTML, CSS, JavaScript) with data processing (Python / ML models) to provide meaningful insights that help students understand placement patterns and make informed career decisions.

Key Features

Interactive dashboards (graphs & charts)
Placement trend analysis (year-wise)
Company-wise recruitment insights
Department performance comparison
(Optional) ML-based prediction of placement chances
Clean UI with modern glassmorphism / dashboard design
